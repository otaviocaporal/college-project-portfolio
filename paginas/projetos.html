MEUS PROJETOS

Estes projetos foram feitos durante o curso de ADS, e foram feitos com HTML e CSS usando o Visual Studio Code, como uma forma de aprender a trabalhar com essas tecnologias e saber como funciona o desenvolvimento web. Inclusive, por enquanto, este portifólio foi o meu maior projeto até agora (abril de 2025).

Claro que meu objetivo é realizar mais projetos no futuro, sem ser somente os do curso, assim poderei entender melhor o mercado e ter uma prática real de desenvolvimento web.

PROJETO DO MEU CURRÍCULO

Este currículo foi o meu primeiro projeto terminado utilizando HTML e CSS. Ele foi feito para mostrar um pouco sobre minha história, minha personalidade e minhas características. Apesar de ter sido um projeto simples, fiquei muito feliz com o resultado e com as features que consegui implementar, como os botões e os links de referência.

Veja uma tabela mostrando os principais comandos utilizados:

Comando Função
<h2> para os títulos e tópicos ficarem em destaque
<ul> e <li> para criar listas e menus
<background-color> (css) para mudar a cor de fundo de textos e divs
<img> para inserir imagens
<a> para inserir links

PROJETO DE TABELA

Esse foi meu primeiro projeto em que usei o <table> para criar uma tabela. Foi um projeto bem simples, mas fiquei muito feliz com o resultado. Me ajudou demais a entender o uso de colunas e linhas para a realização de listas, menus e as próprias tabelas. É uma tabela simples e somente visual, sem interação com botões.

Veja uma tabela mostrando os principais comandos utilizados:

Comando Função
<table> para criar uma tabela
<th> , <tr> e <td> para criar cabeçalhos, linhas e colunas
<background-color> (css) para mudar a cor de fundo
<img> para inserir imagens
<font-weight: bold> (css) para deixar o texto em negrito
